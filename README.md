
# 图书购物系统（Vue.js）

> A Vue.js project

## 启动项目
``` bash
#克隆项目
git clone https://github.com/linchengzzz/book.git

# 安装npm包
yarn / npm install

# 将 ./db/BookSystem.sql 数据库导入MySQL 
# 在./server/mysql.js 文件中设置mysql用户链接

# 启动Node服务器
yarn server / npm server

# 启动项目，访问localhost:8080
yarn start / npm start

```
